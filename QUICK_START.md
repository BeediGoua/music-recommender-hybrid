# Guide de Démarrage Rapide - Music Recommender AI

##  Lancement Ultra-Rapide (2 minutes)

### Option 1: Lancement Automatique (Recommandé)
```bash
# Script tout-en-un qui configure et lance l'application
python launch_premium.py
```

### Option 2: Lancement Manuel
```bash
# 1. Créer les modèles
python create_models_manual.py

# 2. Lancer l'application premium
streamlit run app/streamlit_app_premium.py
```
